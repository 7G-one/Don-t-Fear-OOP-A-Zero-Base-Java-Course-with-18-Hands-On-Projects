> 从 System.out.println 到搭建完整的图书管理系统

## 这是什么?

一套给**想学 Java 但被"面向对象"吓到**的人准备的课程。

不是那种"我给你看代码你自己悟"的课——每节课都有**生活类比、逐行注释、练习题和参考答案**，18 课带你从 `Hello World` 走到能自己写一个完整的图书管理系统。

每课都有"恭喜完成"的正反馈，每个概念都有生活类比帮你理解。学到最后你会明白：为什么 Java 是世界上使用最广泛的编程语言之一。

## 怎么跑？

## 几个小建议

1. **别跳课** — 后面的课用到前面的知识，跳了会懵
2. **别光看** — 编译运行，改参数，看结果怎么变
3. **做练习** — 练习题不是摆设，先自己写再看答案
4. **理解面向对象** — Java 的灵魂是 OOP，多画类图
5. **注意编译错误** — Java 编译器的错误提示很详细，认真读就能看懂

## Java 核心概念

### 面向对象
- **类（Class）** = 图纸，**对象（Object）** = 实物
- **继承** = 遗传，**多态** = 同一个动作不同表现
- **接口** = 合同，签了就必须履行

### 强类型
- 每个变量必须声明类型
- 类型在编译时检查，减少运行时错误

### 跨平台
- 一次编写，到处运行
- JVM（Java 虚拟机）负责执行
